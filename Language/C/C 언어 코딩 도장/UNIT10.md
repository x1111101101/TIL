## 상수와 리터럴
코드의 의도 명확화 - 실수로 값의 변경을 방지 하는 등
```
const int a = 123;

123: 리터럴 - 값 그 자체
a: 상수 - 변수와 같이 리터럴이 저장되는 곳
```


|형태|서식지정자|리터럴 예|출력 예|
|---|----|-----|----|
|8진수|\o|017|017|
|16진수|\h|0x1A|0x1A|
|지수 표기|%e|1.0E-5F|1.0E-5|

### 리터럴 접미사
0x1f, 0.3LLU 등과 같이 표기 -> 자료형 결정
```
1L - long
1.0L - long double
```
