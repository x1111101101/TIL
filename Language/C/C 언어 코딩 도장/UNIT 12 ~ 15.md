# UNIT 13. 증감 연산자 사용하기
부록 17 시퀀스 포인트 참조
``` C
#include <stdio.h>

int main(int argc, char* argv[]) {
	int a = 10, b = 10;
	printf("%d %d\n", a++, b--);
	printf("%d %d\n", a, b);
	return 0;
}

/*
Result:
10 10
11 11
*/
```
# UNIT 14. 곱셈, 나눗셈하기
### 머신 앱실론
부동 소수점 오차의 상한선. (자세한건 UNIT 85. 실수 자료형의 오차에서)

```
#include <stdio.h>
#include <float.h>
#define _CRT_SECURE_NO_WARNINGS
 
int main() {
	float a = 1.0f;
	
	float b = 0.1f;
	printf("%.24f\n", b); // 0.100000001490116119384766
	for(int i = 0; i<10; i++) {
		a += b;
	}
	printf("%d\n", 2.0f == b); // 0
	printf("%.24f \n", FLT_EPSILON); // 0.000000119209289550781250
	printf("%.24f \n", a); // 2.000000238418579101562500
}  
```
